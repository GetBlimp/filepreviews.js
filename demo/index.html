FilePreviews.io Demo

After you submit a file URL or upload a file FilePreviews.io will process the file and generate image previews and a lot of other metadata depending on the file type.

For some files like PSD we can extract previews from all layers and even text. For PDF or Word documents we can extract all pages and all the text. For image files we can even do OCR.

Throw a file at us and see what happens.

FilePreviews.io

Generate image previews and metadata from almost any kind of file


This is a simple interactive demo. Feel free to edit the code.

For this demo we are using filepreviews.js a very simple client library for the FilePreviews.io service. This library will allow any client side application to generate file preview images and metadata without a server.

For some files like PSD we can extract previews from all layers and even text. For PDF or Word documents we can extract all pages and all the text. For image files we can even do OCR.
